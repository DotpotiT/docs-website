Installation

API Setup

  • Open another terminal in the website directory.
  • Go to api directory : cd api
  • To install the dependencies, run the following command: npm install
  • Install Nodemon by running the following command: npm install nodemon
  • Start the server by running the following command: npm start
  • The api is running on at [localhost:8800]http://localhost:8800

Client-side Setup

  • Open another terminal in the website directory.
  • Go to api directory : cd client
  • To install the dependencies, run the following command: npm install --legacy-peer-deps
  • Start the server by running the following command: npm run dev
  • The api is running on at [localhost:5173]http://localhost:5173

Client-side Setup

  • Open another terminal in the website directory.
  • Go to api directory : cd admin
  • To install the dependencies, run the following command: npm install
  • Start the server by running the following command: npm run dev
  • The api is running on at [localhost:5174]http://localhost:5174

Messenger Chat Setup


  • Go to your Page's Settings and click Messaging.
  • Under Add Messenger to your website, click Get started.
  • From here, you can either:
    • Install manually by clicking Set up. From here, you can select your language, add your website domain(s) and copy the code to paste on your website.
    • Follow the Add Chat Plugin to your website steps below based on your chosen website builder and paste the code on the page(s) where you want the Chat Plugin to appear.
  • Under Start a conversation, you can create a welcome message and add automated responses for frequently asked questions.
  • Under Customise your chat plugin options, you can customise the colour, alignment and expansion settings.

After you've completed the chat plugin setup from your Page, Past the code in your client folders index.html file, Before the closing of the body tag .

Admin App Deployment

The Success Pro Theme Admin App is a feature-rich and user-friendly application designed to manage and oversee the Success Pro theme for websites. It provides a modern and intuitive interface that streamlines the management of various website elements. The app offers a wide range of functionalities, including secure API integration for efficient data storage and management, portfolio and contact sections, blog article creation and editing, company information, services and products management, order tracking, job portal administration, and website theme customization. Powered by cutting-edge technologies such as Node.js, React Vite, MongoDB, and Tailwind CSS, the app ensures a seamless and responsive experience across different devices. By deploying the Success Pro Theme Admin App on Vercel, website administrators can easily host and access the app, making it an all-in-one solution for effective website administration and customization.

Customize Website

Customizing a website from the admin panel allows website owners or administrators to make changes to their website without needing to access the website's code directly. With an admin panel, users can customize various aspects of their website, including the design, layout, content, and functionality. They can add or remove pages, update menus, change themes, adjust color schemes, and much more, all through a user-friendly interface. This type of customization allows website owners to make quick and easy changes to their website, ensuring that their site remains up-to-date and responsive to their audience's needs. Overall, the admin panel provides a convenient and efficient way to manage and customize a website without requiring technical expertise.
